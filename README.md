# 📊 Coffee Shop Sales Analysis Dashboard (Excel Project)

## 📌 Project Overview
This project focuses on analyzing coffee shop sales data using Microsoft Excel.  
The goal is to derive meaningful insights and build an interactive dashboard to support business decision-making.

---

## 🎯 Objectives
- Analyze sales performance across different time periods
- Identify peak hours and customer behavior
- Compare store locations and product performance
- Create a dynamic and interactive dashboard

---

## 🗂️ Dataset Description
The dataset includes:
- Transaction ID
- Date & Time
- Store Location
- Product Category & Type
- Quantity Sold
- Unit Price

---

## ⚙️ Data Cleaning & Transformation
Performed using **Power Query**:
- Removed unnecessary text (LG, RG, SM)
- Created a new **Size column** (Small, Regular, Large)
- Trimmed extra spaces
- Converted data types
- Extracted:
  - Month Name
  - Day Name
  - Hour

---

## 🧮 Feature Engineering
- Created **Total Bill = Unit Price × Quantity**
- Derived time-based insights for better analysis

---

## 📊 Data Analysis
Used **Pivot Tables & Power Pivot** to:
- Calculate total sales and orders
- Analyze hourly sales trends
- Measure customer footfall
- Compare location-wise performance

---

## 📈 Dashboard Features
- Interactive filters (Slicers):
  - Month
  - Day of Week
  - Hour
- KPIs:
  - Total Sales
  - Total Orders
  - Average Bill per Person
- Visualizations:
  - Line Chart (Sales Trends)
  - Bar Charts (Category & Location Analysis)

---

## 🔍 Key Insights
- Peak sales occur during morning hours
- Certain locations outperform others consistently
- Customer purchase behavior varies by time of day
- High demand for specific product categories
